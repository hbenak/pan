# PAN
Landing page e template de email responsivo

## Dependências:
- Node (10.15.0);
- Bootstrap (4.1.1);
- Font Awesome (4.7.0);
- Jquery (3.3.1);
- Node Sass (4.11.0);
- Webpack (4.8.3);
- Webpack CLI (2.1.3);
- Webpack Dev Server (3.1.11).

## Padronização de código:
- Sublime Text;
- VS Code;
- Atom;
- Outros.

## Instalação:
Na pasta /landing-page executar:
$ npm install

### Modo de desenvolvimento
$ npm start
Acessar em **localhost:8080**

### Modo de produção
$ npm run build
